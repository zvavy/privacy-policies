Privacy Policy

Last updated: 23.09.2024

Welcome to WordSensei. Your privacy is important to us. This privacy policy explains how we collect, use, and protect your data while using our Chrome extension.

Information We Collect

WordSensei does not collect any personal information. However, we use anonymous usage data for analysis purposes to improve the extension's performance and features.

Permissions

WordSensei requires the following permissions to function:

  • Active Tab: This allows WordSensei to access and analyze content on the current webpage to provide definitions, synonyms, and examples.
  • Context Menu: Enables the ability to define words using the right-click menu.
  • Storage: This allows WordSensei to save user settings and preferences via Supabase, ensuring a customized experience across sessions.
  • Scripting: Facilitates communication between the background and content scripts for seamless functionality.

Third-Party Services

WordSensei uses the ChatGPT API for its AI-powered definitions, synonyms, and sentence examples. No personal data is shared with this service; only the text you select for analysis is sent for processing.

Data Storage

User settings and preferences are stored via Supabase. No personal or sensitive data is stored, and all stored information is anonymized for security purposes.

Data Security

We take data security seriously and employ measures like regular backups and anonymization to protect any stored information. Your data is never shared with third parties.

Changes to This Privacy Policy

We may update this privacy policy from time to time. Any changes will be posted here with the updated date.
